# Reference teardowns — the taste benchmark

Operator-curated premium sites, reverse-engineered. The design + build stages MUST
study this and match this level. (Add more over time — more references = better taste.)

## Cross-cutting premium patterns (use these)
- **Radius varies by brand** — premium is often SHARP (0–4px) or very subtle, NOT the
  default rounded-2xl everywhere. Pick a deliberate shape language per brand:
  metawrapz/ohhhcontest = sharp; pem-tech = 16–24px; justluxury = 8–12px. **Never reuse one radius for all sites.**
- **Dark sophisticated palettes** for luxe/automotive/creative (charcoal/near-black + one accent);
  **warm cream/earth** for travel/hospitality. High contrast, restrained.
- **Trust density**: repeated review badges (4.9/5 from 55+), client logo strips, partner logos,
  specific metrics (+22 years, 1,200+ travelers, 2,000+ trips), real photography (never stock-looking).
- **Numbered process steps** (001–006), often sticky/pinned as you scroll.
- **Horizontal-scroll carousels**: fleet, winners, gallery, portfolio — with arrow nav or drag.
- **Floating contact CTA**: phone (tel:) or WhatsApp, persistent.
- **Sticky widgets**: booking/search bar stays accessible.
- **Decorative SVG corner frames** on images (ohhhcontest), texture/grain overlays.
- **Whitespace discipline + oversized bold headlines** with confident type pairing.

## Per-site notes
- **pem-tech.fr** (B2B multi-technical, Framer/Webflow): navy/charcoal, sans, rounded 16–24px cards,
  services grid → case studies w/ metrics → client logos. Authority via scale + partners (ENGIE/Vinci).
- **thrivedoc.co** (high-ticket medical marketing, **Framer**): deep navy, minimal radius, 3-card problem
  framework, stat callouts, case-study carousel, 5-step process, FAQ accordion, exclusivity positioning.
- **ohhhcontest.com** (filmmaker contests, **Webflow**): black, bold anti-corporate headlines, numbered
  steps 001–006, winner horizontal carousel, parallax stacked photos, SVG corner frames, sharp corners.
- **safarai.ma** (AI Morocco travel): warm neutrals + ochre/terracotta, minimal radius 8–12px, single-prompt
  hero, 3-step cards, destination cards w/ metadata, testimonial 3-col grid, FAQ, waitlist scarcity.
- **metawrapz.co** (car wraps/PPF, **Framer**): near-black + saturated car imagery + gold micro-accents,
  sharp corners, dark hero + single CTA, looping portfolio carousel, repeated 4.9/5 badges, floating
  "Free Estimate" tel: button, "why us" stacked imagery. Luxe minimalism + motorsport.
- **spenergetique.fr** (B2B energy compliance): navy/charcoal, sans, tight radius, 6-step vertical process,
  audience segmentation grid, 3 pillar cards, blog feed. Form-follows-function authority.
- **justluxurytransfers.com** (Agadir luxury transfers, **Sanity + Next.js**): charcoal + warm gold + sand,
  8–12px radius, full hero + sticky booking widget, fleet horizontal carousel w/ specs, route cards w/
  pricing, excursion masonry, CEO quote, Google reviews carousel, 5 languages, persistent WhatsApp.

## The takeaway for the factory
Premium = a DELIBERATE, brand-specific shape+motion+layout language (not a recolored template) +
trust density + real photography/film + restraint. Match the relevant teardown for the client's niche.
